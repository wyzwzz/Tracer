# Tracer
My offscreen rendering lab based on PBRT

## Features
### Renderer
- [ ] Whitted
- [ ] DirectLighting
- [x] Path
- [ ] BDPT
- [ ] MLT
- [ ] AO

### Material
- [x] Phong
- [x] Glass
- [ ] Homogeneous/heterogeneous medium
- [ ] Metal
- [ ] Plastic
- [ ] Substrate?
- [ ] Disney

### Light
- [x] Diffuse
- [x] IBL
- [ ] Spot
- [ ] Point
### Others
- [x] SAH based BVH
- [x] Depth of field
- [x] Film filter
- [x] Multiple importance sampling
- [ ] Tone mapping
- [ ] BSSRDF
- [ ] SIMD
